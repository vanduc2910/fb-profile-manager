# FB Profile Manager

## Project Description
FB Profile Manager is a comprehensive tool designed to help users manage their Facebook profiles effectively. This application allows users to organize their profile information, enhance their online presence, and improve privacy settings with ease.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/vanduc2910/fb-profile-manager.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd fb-profile-manager
   ```
3. **Install the required dependencies:**
   ```bash
   npm install
   ```
4. **Run the application:**
   ```bash
   npm start
   ```

## Feature Overview for PHASE 1
- **User Authentication**: Secure login for users to protect their profile data.
- **Profile Management**: Ability to update and manage various personal information fields.
- **Privacy Controls**: Options to modify privacy settings for profile visibility.
- **Activity Log**: A log that shows recent changes made to the profile for user tracking.
